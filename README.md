# Pro Controller Tester (procon-tester)

Nintendo Switch Proコントローラー（およびその他のゲームパッド・ジョイスティック）の入力状態を確認するためのテストツールです。Pygame (pygame-ce) を利用して、コントローラーからの入力を画面上、またはコンソール上でリアルタイムに確認できます。

## ファイル構成

- **`main.py`**
  - GUI版のテスターです。ウィンドウ上にボタンや十字キーの入力状態、左右のアナログスティックの座標、トリガーの押し込み具合などを視覚的・グラフィカルに表示します。
- **`b.py`**
  - CLI版のシンプルなテスターです。各ボタンの押下状態やスティック（Axis）の値を、ターミナル上にテキストとして連続出力します。

## 必要要件

- Python 3.12 以上
- [pygame-ce](https://pyga.me/) (>=2.5.6)
- pyglet (>=2.1.13) ※`pyproject.toml`記載

## セットアップ手順

`pyproject.toml` が用意されているため、`pip` や `uv` などのパッケージマネージャで簡単にインストールできます。すでに `.venv` などの仮想環境がある場合は、そちらを有効化してご利用ください。

```bash
# 仮想環境の有効化（例: macOS/Linux の場合）
source .venv/bin/activate

# 依存パッケージのインストール
pip install -e .
```

## 使い方

1. Proコントローラー（またはゲームパッド）をPCに接続します（BluetoothまたはUSB有線）。
2. プログラムを実行して入力を確認します。

**GUIでグラフィカルに確認する場合:**
```bash
python main.py
```

**ターミナル上で数値として確認する場合:**
```bash
python b.py
```

## 備考
- 環境（OSやドライバ）によって、取得できるボタンの数が変わったり、ボタンのマッピング（インデックス）が異なる場合があります（特にMacでの接続時）。
